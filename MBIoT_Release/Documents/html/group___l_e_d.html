PUBLIC u16 APP_LED_Close ( u8  ledMask)

Free LED control, occupied GPIO returns to digital input with internal pull high.

Parameters
[in]ledMask
Returns
MBIOT_RES_SUCCESS Successfully free LED bus.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Example:
Close LED1.
APP_LED_Close(LED1_MASK);
PUBLIC u16 APP_LED_Combo_Flash ( APP_LED_ConfParas XDATA *  p_profile)

Configure lighting parameters of LED1 and LED2, starts to flash both LEDs.

Parameters
[in]p_profilePointer to APP_LED_ConfParas structure.
Returns
MBIOT_RES_SUCCESS Successfully configure lighting parameters of LED1 and LED2, starts to flash both LEDs.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Example:
Interleave both LEDs flashes 3 tiimes within 2 minutes, each time turn on 100ms and turn off 50ms .
XDATA APP_LED_ConfParas led;
led.flashType=LED_TYPE_COMBO_FLASH_INTERLEAVE;
led.onInterval=LED_DURATION_100_MS;
led.offInterval=LED_DURATION_50_MS;
led.count=LED_TIMES_3;
led.flashInterval=(LED_DURATION_1_SEC*2);
APP_LED_Combo_Flash(&led);
PUBLIC u16 APP_LED_Init ( u8  ledMask)

LED initialization.

Parameters
[in]ledMaskMask of LEDs. If LED0_MASK is set, the P02 is occupied. If LED1_MASK is set, the P35 is occupied. See LED_BIT_MASK.
Returns
MBIOT_RES_SUCCESS Successfully initialz LED.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Example:
Initial LED0 and LED1.
APP_LED_Init(LED0_MASK|LED1_MASK);
PUBLIC u16 APP_LED_LED0_Flash ( APP_LED_ConfParas XDATA *  p_profile)

Configure lighting parameters of LED0 and start to flash.

Parameters
[in]p_profilePointer to APP_LED_ConfParas structure.
Returns
MBIOT_RES_SUCCESS Successfully configure lighting parameters of LED0 and starts to flash.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Example:
LED0 flashes 2 tiimes within 1 minute, each time turn on 50ms and turn off 50ms .
XDATA APP_LED_ConfParas led;
led.flashType=LED_TYPE_SINGLE_FLASH;
led.onInterval=LED_DURATION_50_MS;
led.offInterval=LED_DURATION_100_MS;
led.count=LED_TIMES_2;
led.flashInterval=LED_DURATION_1_SEC;
APP_LED_LED0_Flash(&led);
