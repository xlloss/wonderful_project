Member Data Documentation

u8 APP_UART_ConfParas::baudrate

UART baud rate, see UART_BAUDRATE_DEF.

u8 APP_UART_ConfParas::flowContol

UART flow control option, see UART_FLOW_CONTROL_DEF.
If the parameter is set to APP_UART_FLOW_CONTROL_ENABLE, then the GPIO P00 is occupied for UART_CTS and the GPIO P36 is occupied for UART_RTS.

u8 XDATA* APP_UART_ConfParas::rxBuffer

Pointer to the UART RX buffer.

u16 APP_UART_ConfParas::rxBufferSize

UART RX buffer size, see the minimum limition APP_UART_BUFFER_MIN_LENGTH.

u16 APP_UART_ConfParas::rxNotifyLength

The length of queued data in UART RX buffer needs to be notified by UART callback event APP_UART_EVT_RECEIVE_DATA.

u8 APP_UART_ConfParas::rxWakeupPinNum

The pin number used to wake up MBIoT from low power mode to receive UART data, see GPIO_PIN_NUM_DEF

u8 XDATA* APP_UART_ConfParas::txBuffer

Pointer to the UART TX buffer.

u16 APP_UART_ConfParas::txBufferSize

UART TX buffer size, see the minimum limition APP_UART_BUFFER_MIN_LENGTH.

u8 APP_UART_ConfParas::txIndicateInterval

The timing setting of UART_Tx_Ind signal deasserted if txIndicateInterval is not zero. UART_Tx_Ind is low before UART_Tx packet is transmitted.
The GPIO P02 is occupied as UART_Tx_Ind if the value is not set to 0. (Unit: 625us)
