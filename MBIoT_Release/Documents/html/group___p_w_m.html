PUBLIC u16 APP_PWM_Close ( u8  pwmMask)

Free PWM channel bus, the occupied GPIO pins return to input and internal pull high.

Parameters

Returns
MBIOT_RES_SUCCESS Successfully free PWM channel bus.

APP_PWM_Close(PWM1_MASK|PWM2_MASK);
PUBLIC u16 APP_PWM_Config ( u8  pwmMask,
APP_PWM_ConfParas XDATA *  p_pwmParas 
)

Initialize PWM Channel.

Parameters
[in]pwmMaskThe mask of PWM channel.
If PWM0_MASK is set, the P21 is occupied. If PWM1_MASK is set, the P22 is occupied.If PWM2_MASK is set, the P23 is occupied. If PWM3_MASK is set, the GPIO P25 is occupied.
[in]p_pwmParasPointer to the APP_PWM_ConfParas structure.
Returns
MBIOT_RES_SUCCESS Successfully configuring PWM Channel.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Example:
XDATA APP_PWM_ConfParas pwm = {PWM_CLOCK_32K, 319, 159, POSITIVE_OUTPUT};
APP_PWM_Config((PWM0_MASK), &pwm);
The result of PWM0 is :
Frequency=clockSource/(autoReloadValue+1)=32000/(319+1) = 100 Hz
Duty cycle=((autoReloadValue-compareValue)/(autoReloadValue+1))* 100% = (319-159)/(319+1)=50%
PUBLIC u16 APP_PWM_Start ( u8  pwmMask)

Start PWM signal by specified channel(s).

Parameters
[in]pwmMaskThe mask of PWM channel.
Returns
MBIOT_RES_SUCCESS Successfully start the PWM.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
PWM must be initialized befor start PWM signal. And the parameters of PWM can't be changed during PWM signal is actived.
PWM signal must be stoped before changing the parameters of PWM.
Example:
Start PWM couner for PWM channel 1.
APP_PWM_Start(PWM1_MASK);
