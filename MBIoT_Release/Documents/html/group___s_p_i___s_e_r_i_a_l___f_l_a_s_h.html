PUBLIC u16 APP_SPI_SerialFlash_Read ( APP_SPI_SFlashRParas XDATA *  p_readCmd)

Get data from SPI serial flash.

Parameters
[in]p_readCmdPointer to APP_SPI_SFlashRParas structure.
Returns
MBIOT_RES_SUCCESS Successfully read data from Serial Flash memory.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Supporting eight-clocks command in Standard SPI mode for Serial Flash memory.
SPI mode 0 and 16MHz clock frequency are used as default settings.
To drive Read Instructions of Serial Flash. Example:
Read 7 bytes data from seiral flash address 0x000006.
Total length of instruction content is 0x03.
Read Instruction is 0x03.
XDATA APP_SPI_SFlashRParas readCmd;
readCmd.contentEntry[0]=0x00;
readCmd.contentEntry[1]=0x00;
readCmd.contentEntry[2]=0x06;
readCmd.contentLen = 0x03;
readCmd.opcode = 0x03;
readCmd.readLen = 7;
APP_SPI_SerialFlash_Read(&readCmd);
PUBLIC u16 APP_SPI_SerialFlash_Write ( APP_SPI_SFlashWParas XDATA *  p_writeCmd)

Write data into SPI serial flash.

Parameters
[in]p_writeCmdPointer to APP_SPI_SFlashWParas structure.
Returns
MBIOT_RES_SUCCESS Successfully read data from Serial Flash memory.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Supporting eight-clocks command in Standard SPI mode for Serial Flash memory.
SPI mode 0 and 16MHz clock frequency are used as default settings.
To drive Register/Setting Instructions of Serial Flash.
Example:
Write 4 bytes length of data (0x08,0x09,0x0a,0x0b) in seiral flash address 0x000006.
And the Write Enable instruction is 0x06.
Page Program Instructions is 0x02.
Total length of instruction content is 0x07.
XDATA APP_SPI_SFlashWParas writeCmd; writeCmd.writeEnable = 0x06; writeCmd.contentLen = 0x07; writeCmd.opcode = 0x02; writeCmd.contentEntry[0]=0x00;
writeCmd.contentEntry[1]=0x00;
writeCmd.contentEntry[2]=0x06;
writeCmd.contentEntry[3]=0x08;
writeCmd.contentEntry[4]=0x09;
writeCmd.contentEntry[5]=0x0a;
writeCmd.contentEntry[6]=0x0b;
APP_SPI_SerialFlash_Write(&writeCmd);
