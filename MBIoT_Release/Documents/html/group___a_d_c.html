PUBLIC u16 APP_ADC_GetValue ( u8  channel,
u16 XDATA *  p_readValue

Get current ADC value(12 bits resolution, and 10 bits ENOB) of specified ADC channel.

Parameters
[in]channelSpecify which ADC channel needs to be readed, see ADC_CHANNEL_NUM.
[out]p_readValuePointer to the value got via specified channel.
Returns
MBIOT_RES_SUCCESS Successfully get a value from specified ADC channel.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Exampe:
Get ADC value via channel 1.
XDATA u16 read;
APP_ADC_GetValue(ADC_CHANNEL1, &read);
