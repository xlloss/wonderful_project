Detailed Description

The attribute value part definition of one attribute. The value format is for 16-bit Include Service definition. The Service UUID shall only be present when the UUID is a 16-bit Bluetooth UUID. Core Spec. 4.2 vol.3, Part G, Sec. 3.2.
