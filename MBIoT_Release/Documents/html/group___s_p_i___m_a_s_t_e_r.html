PUBLIC u16 APP_SPI_MASTER_SendReceive ( APP_SPI_Master_SendReceParas XDATA *  p_spiMasterSendRece)

Start SPI communication.

Parameters
[in]p_spiMasterSendRecePointer to the APP_SPI_Master_SendReceParas structure.
Returns
MBIOT_RES_SUCCESS SPI bus communicate successfully.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Hardware SPI master mode can connect Two SPI slaves by spiMasterId setting.
If SPI_MASTER_HW_NCS0 is set, below GPIO pins are occupied for SPI bus.
  • P31: NCS
  • P32: MISO
  • P33: MOSI
  • P34: SCLK
    If SPI_MASTER_HW_NCS1 is set, below GPIO pins are occupied for SPI bus.
  • P27: NCS
  • P32: MISO
  • P33: MOSI
  • P34: SCLK
    If SPI_MASTER_MANUAL_NCS is set, below GPIO pins are occupied for SPI bus.
  • User define PXX for NCS
  • P32: MISO
  • P33: MOSI
  • P34: SCLK
