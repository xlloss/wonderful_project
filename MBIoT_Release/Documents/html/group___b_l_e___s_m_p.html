Enumeration Type Documentation

enum BLE_SMP_EventID

BLE SMP callback events.

Enumerator
BLE_SMP_EVT_PAIRING_COMPLETE 

Pairing process is finished. See the BLE_SMP_Evt_PairingComplete structure for eventField.

BLE_SMP_EVT_SECURITY_REQUEST 

Receives Pairing request from remote device. See the BLE_SMP_Evt_SecurityReq structure for eventField.

BLE_SMP_EVT_NUMERIC_COMPARISON_CONFIRM_REQUEST 

Passkey is needed to be confirmed for pairing process. See the BLE_SMP_Evt_DisplayCompareValue structure for eventField.

BLE_SMP_EVT_INPUT_PASSKEY 

Passkey is requested to be inputted for pairing process. See the BLE_SMP_Evt_InputPasskey structure for eventField.

BLE_SMP_EVT_DISPLAY_PASSKEY_REQUEST 

Passkey is needed to be displayed for pairing process. See the BLE_SMP_Evt_DisplayPasskeyRequest structure for eventField.

Initiate pairing procedure.

Note
For master role, BLE SMP starts the Pairing Feature Exchange by sending a Pairing Request. command to the responding device. For slave role, BLE SMP request that the master initiates security with the requested security properties.
Parameters
[in]connHandleConnection handle.
Returns
MBIOT_RES_SUCCESS Initiate pairing procedure successfully.
MBIOT_RES_FAIL The specified connection handle is invalid.
