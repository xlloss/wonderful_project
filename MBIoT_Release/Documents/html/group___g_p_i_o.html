PUBLIC u16 APP_GPIO_Pin_Clear ( u8  pinNum)

Set a specified GPIO pin to LOW.

Parameters
[in]pinNumPin number, GPIO_PIN_NUM_DEF.
Returns
MBIOT_RES_SUCCESS Successfully clear a GPIO pin.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
The direction of the specified GPIO pin must be configured to output before this API is executed.
Example:
Set P0_1 to LOW.

PUBLIC u16 APP_GPIO_Port_Clear ( u8  portNum,
u8  clearMask 
)

Set multiple GPIO pins within a PORT to LOW.

Parameters
[in]portNumPort number, see GPIO_PORT_DEF.
[in]clearMaskBit mask of the specified GPIO port, see GPIO_PORT_BIT_MASK. The GPIO pin(s) can be set to LOW only when the bit mask is configred. And the other GPIO pin(s) will not be influenced.
Returns
MBIOT_RES_SUCCESS Successfully set mutilple GPIO pins within a PORT to LOW.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
The direction of the specified GPIO port must be configured to output before this API is executed.
Example:
Set GPIO port3 bit 5(P35) and bit7(P37) to LOW.
APP_GPIO_Port_Clear(GPIO_PORT3, GPIO_PORT_MASK_BIT5|GPIO_PORT_MASK_BIT7);
