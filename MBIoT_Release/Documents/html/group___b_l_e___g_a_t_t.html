 ATT_PERMISSION_KEY_SIZE
 Attribute might requires permission(Encryption/Authentication Permission) to accessed. This key size defines the minimum length of the pairing key. Shorter than this length will be taken as un-safe authentication/encryption. Length ranges from 7 bytes to 16 bytes. Set to zero to disable the key size checking.
 
 GATT_PROCEDURE_STATUS
 Some GATT procedures would not finish in one ATT request. Thus, more than one event will receive in one procedure. Status in the event indicates that if this procedure will continue or end.

The Maximum supported MTU size of ATT stack.

PUBLIC u16 BLE_GATTC_DiscoverAllCharacteristics ( u8  connHandle,
u16  startHandle,
u16  endHandle 
)

GATT Client initialize the discovering all characteristics procedure with the given handle range. Two possible response events will come for this procedure. Receive BLE_GATTC_EVT_DISC_CHAR_RESP if characteristics are discovered. Status flag indicates if more events are coming or procedure end. Receive BLE_GATTC_EVT_ERROR_RESP when no characteristic is found. The procedure also ends.

Parameters
[in]connHandleHandle of the connection to discover the services.
[in]startHandleStart of the querying handle range.
[in]endHandleEnd of the querying handle range.
Returns
MBIOT_RES_SUCCESS Successfully starts the discovering.
MBIOT_RES_INVALID_PARA Invalid parameters.
