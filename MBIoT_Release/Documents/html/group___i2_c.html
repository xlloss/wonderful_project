PUBLIC u16 APP_I2C_CombineWR ( APP_I2C_CombineWRParas XDATA *  p_combineWRParas)

Send data to I2C slave side and then get data from I2C slave side.

Parameters
[in]p_combineWRParasPointer to APP_I2C_CombineWRParas structure.
Returns
MBIOT_RES_SUCCESS Successfully send data to I2C slave side.
I2C_ERR_NO_CONFIG I2C bus is not initialized.
I2C_ERR_ADDR_NACK Receives a NACK on address.
I2C_ERR_LENGTH_EXCEEDED Data length is exceeded.
Note
Example:
Send 20 bytes length of data to I2C slave side, and then get 10 bytes length of data form I2C slave.
XDATA APP_I2C_CombineWRParas p_combineWR;
p_combineWR.lenWrite = 20;
xmemset(&p_combineWR.writeBuffer, 0x11, p_combineWR.lenWrite);
p_combineWR.lenRead = 10;
xmemset(&p_combineWR.readBuffer, 0, p_combineWR.lenRead);
APP_I2C_CombineWR(&p_combineWR);
PUBLIC u16 APP_I2C_Config ( u8  address,
u8  speed 
)

I2C configuration.

Parameters
[in]address7-bit address of the Slave.
[in]speedI2C bus speed, see I2C_BUS_SPEED.
Returns
MBIOT_RES_SUCCESS Configuring I2C successfully.
MBIOT_RES_INVALID_PARA Invalid parameters.
Note
Two GPIO pins are occupied if I2C bus are enabled.
I2C SCL is pin P12, I2C SDA is pin P13.
Example:
Access E2prom by I2C bus.
EEPROM(I2C slave) address is 0xA0. I2C bus speed is 200KHz.
APP_I2C_Config(0xA0, DIV_TO_200_KHz);
