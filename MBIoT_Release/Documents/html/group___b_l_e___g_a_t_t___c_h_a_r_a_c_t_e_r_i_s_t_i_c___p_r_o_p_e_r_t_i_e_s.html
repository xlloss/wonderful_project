#define AUTHENTICATED_SIGNED_WRITES   0x40

If set, permits signed writes to the Characteristic Value.

#define BROADCAST   0x01

If set, permits broadcasts of Characteristic Value using Server Characteristic Configuration Descriptor.

#define EXTENDED_PROPERTIES   0x80

If set, additional characteristic properties are defined in the Characteristic Extended Properties Descriptor.

#define INDICATE   0x20

If set, permits indications of Characteristic Value with acknowledgement.

#define NOTIFY   0x10

If set, permits notifications of a Characteristic Value without acknowledgement.

#define READ   0x02

If set, permits reads of the Characteristic Value.

#define WRITE   0x08

If set, permits writes of Characteristic Value with response.

#define WRITE_WITHOUT_RESPONSE   0x04

If set, permits writes of Characteristic Value without response.
